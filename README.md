# IKNA Cards

Веб-приложение для интервального повторения учебного материала с помощью карточек.

## Описание

IKNA Cards помогает пользователю изучать материал через карточки с вопросами и ответами. Пользователь может зарегистрироваться, войти в приложение и пройти повторение учебной колоды. Во время повторения пользователь оценивает сложность карточки, а приложение рассчитывает следующий интервал повторения.

На текущем этапе реализован frontend-прототип и начальная backend-часть с проектированием базы данных. Данные учебной колоды загружаются из JSON-файла через API-слой, чтобы позже было проще подключить backend API.

## Стек

- React
- TypeScript
- Vite
- React Router
- CSS
- PostgreSQL

## Реализовано

- регистрация и вход пользователя;
- dashboard после входа;
- режим повторения карточек;
- адаптивная верстка;
- структура базы данных backend-части;

## Структура

```txt
ikna-front/
  public/mock-api/     # JSON-данные для имитации backend
  src/app/             # провайдеры и роутинг
  src/entities/        # типы предметных сущностей
  src/features/        # бизнес-функции приложения
  src/pages/           # страницы приложения
  src/shared/          # общие UI-компоненты и стили
ikna-back/
  database/            # SQL-схема и начальные данные
  docs/                # описание таблиц, ключей и связей
```

## Backend и база данных

```bash
cd ikna-back
psql -d ikna_cards -f database/schema.sql
psql -d ikna_cards -f database/seed.sql
```

Описание проектирования базы данных: `ikna-back/docs/database-design.md`.

## Запуск frontend

```bash
cd ikna-front
npm install
npm run dev
```
